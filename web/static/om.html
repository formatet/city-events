About

This site collects events from venues across the city into a single calendar. It's an independent hobby project – updated nightly, free to use.

Usage

Everything is sorted chronologically. Click the crystal icon next to a title for more info or ticket purchase. Click a date, venue, or category to filter – click again to clear. Click "When" to pick a specific date from a dropdown.

Data

Event data is freely available as JSON and CSV, updated nightly. Licensed under CC-BY 4.0 – use freely, credit this site as source.

Fork this for your city

Live in a city without something like this? The code is open and built to be forked. All you need to do is write scrapers for your local venues – the database, API, and frontend are generic. See the GitHub repo for setup instructions.

Privacy

No cookies. No personal data. Anonymous visit statistics via Umami.
